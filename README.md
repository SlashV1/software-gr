# Sistema Integral de Gestión - Software FJ

Este proyecto es una solución orientada a objetos para la gestión de clientes, servicios y reservas, desarrollada para el curso de **Programación (213023)** de la UNAD.

## 🚀 Características
- **POO Rigurosa:** Implementación de Abstracción, Herencia, Polimorfismo y Encapsulamiento.
- **Manejo de Errores:** Sistema robusto con excepciones personalizadas y bloques `try/except/finally`.
- **Logs de Eventos:** Registro automático de errores en `sistema_errores.log`.
- **Sin Base de Datos:** Gestión total mediante listas y objetos en memoria.

## 🛠️ Tecnologías
- Python 3.x
- Módulo `logging` para trazabilidad.
- Módulo `abc` para clases abstractas.
